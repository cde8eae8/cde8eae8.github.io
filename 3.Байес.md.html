Битва экстрасенсов

Что мы хотим?

У нас есть последовательность из 0 и 1. Мы хоти по ней понять, с какой вероятностью появляется 0, с какой 1.

Очень темная магия

У нас есть функция plot_beta(n_ones, n_zeros) , которая по числу нулей и числу единиц как-то строит наше "представление" о реальной вероятности выпадения 1.

Например, увидев последовательность 0, 1, 0, 0, 1, 0 мы можем вызвать plot_beta(2, 4) и получить как выглядят наши ожидания (мы ожидаем что вероятность что-то около 2/6) и сомнения о вероятности выпадения нулей и единиц.

Темная магия заключается в том, что мы не знаем как именно функция это строит, но нам это и не надо - мы просто передаем число единиц и число нулей в функцию и получаем график.

Также есть функция plot_beliefs(seq, start). Она вызывает plot_beta(number_of_zeros(prefix) + start[0], number_of_ones(prefix) + start[1]) для каждого префикса последовательности, считая, что изначально (до того как мы видим последовательность) наша информация о вероятности выражается массивом start.

То есть, например, plot_beliefs([1, 0, 1, 0], [3, 6]) вызовет

    plot_beta(3, 6)         # for []
    plot_beta(3 + 1, 6)     # for [1]
    plot_beta(3 + 1, 6 + 1) # for [1, 0]
    plot_beta(3 + 2, 6 + 1) # for [1, 0, 1]
    plot_beta(3 + 2, 6 + 2) # for [1, 0, 1, 0]

Также величина наших сомнений (синяя линия) тем меньше, чем больше n_zeros + n_ones в вызове plot_beta

Первая кровь

  • Получить последовательность длины 100 
    p = 0.42
# Сгенерировать игру из 100 шагов с вероятностью выпадения единицы 0.42
seq = game(0.42, 100)
  • Нарисовать график наших предположений для случаев:
    • нет априорных знаний 
      plot_beta(n_ones(seq), n_zeros(seq))
    • есть подсказка. Например 
      plot_beta(n_ones(seq) + 42, n_zeros(seq) + 58)
    • есть дезинформация 
      plot_beta(n_ones(seq) + 90, n_zeros(seq) + 32)

То есть если есть подсказка, у нас изначально есть информация о верном соотношении нулей и единиц, например 42 единицы и 58 нулей, или 210 и 290. А в случае дезинформации у нас есть неверная информация, тут подойдет что угодно, что не соответствует реальной вероятности.

Python

f'some string with {variables}'

Если перед строкой поставить f, а в строке в фигурных скобках написать код на питоне, он вычислится и вместо скобок подставится значение выражения

f"5! is {1 * 2 * 3 * 4 * 5}" -> "5! = 120"

Можно использовать переменные

seq = [0, 1, 2, 3, 4]
f"seq is {seq}" -> "seq is [0, 1, 2, 3, 4]"

Архивы и молнии

for val1, val2, val3 in zip(arr1, arr2, arr3):
    # some code

Если arr1, arr2 и arr3 одной длины, то val1, val2 и val3 будут принимать значения элементов arr1, arr2, arr3 соответственно. Работает с любым числом массивов.

Например,

numbers = [0, 1, 2, 3, 4, 5]
squares = [0, 1, 4, 9, 16, 25]
for number, square in zip(numbers, squares):
    print(f'{number}^2 = {square}')

# Вывод:
# 0^2 = 0
# 1^2 = 1
# 2^2 = 4
# 3^2 = 9
# 4^2 = 16
# 5^2 = 25

# Абсолютно аналогично следующему коду, но код выше читается лучше:
numbers = [0, 1, 2, 3, 4, 5]
squares = [0, 1, 4, 9, 16, 25]
for i in range(len(numbers)):
    print(f'{numbers[i]}^2 = {squares[i]}')

Больше генераторов

Генераторы - штуки чтобы быстро сделать массив. Самый простой вид:

arr = [ expression for i in some_sequence ]

Внешние скобки можно опускать, но иногда это не работает. В принципе если писать их всегда, вы точно не проиграете, пока массив достаточно маленький

В результате получим массив результатов вычисления expression для каждого элемента some_sequence. Например,

squares = [ i * i for i in range(5) ] 
# [0, 1, 4, 9, 16]
pow4 = [ i * i for i in squares ] 
# [0, 1, 16, 81, 256], то есть еще раз возвели в квадрат каждый элемент squares

# Можно вызывать функции
factorials = [ factorial(i) for i in squares ] 
# [1, 1, 2, 6, 24, 120]

Чуть более расширенная форма:

arr = [ expression for i in some_sequence if expression ]

Делает то же что и штука выше, но если не выполнено expression, то пропускает этот элемент. Очень удобно, например, считать число элементов в массиве, которые удовлетворяют условию

seq = [0, 0, 0, 1, 1, 1, 1, 0, 1, 1, 0]
n_zeros = sum(1 for i in seq if i == 0)
# То есть для каждого 0 добавим в результирующий массив число 1, а потом найдем сумму получившегося массива
# n_zeros == 5

# Аналогично
n_ones = sum(1 for i in seq if i == 1)
# n_ones == 6
