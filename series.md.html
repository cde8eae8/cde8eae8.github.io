Предел

Колдую!

Формальное определение, что предел последовательности \(a_n\) равен A: \[\forall \varepsilon > 0 \exists N : \forall n > N |a_n - A| < \varepsilon \]

Перевод

\(\forall\) означает для любого; \(\exists\) - существует; \(:\) - “такой, что”; \(|x|\) - модуль \(x\), равен \(x\) для \(x \geq 0\), \(-x\) для \(x < 0\).

Получаем “Для любого положительного эпсилон существует \(N\), такой что для любого \(n > N\) выполнено \(|a_n - A|\) меньше эпсилон”

Это “Для любого положительного эпсилон существует \(N\), начиная с которого для каждого элемента \(a_n\) последовательности будет выполнено \(a_n\) отличается от \(A\) не больше чем на эпсилон”

Если еще проще, то какое бы маленькое число эпсилон мы не взяли, начиная с некоторого момента все дальнейшие значения последовательности будут отличаться от A не больше чем на это маленькое число. Для каждого отдельного эпсилон этот начальный момент может быть разным, но факт в том, что он существует.

Полезный факт об оценке

Если \(a \le b\), то \(\frac{1}{a} \ge \frac{1}{b}\)

Если \(\frac{1}{a} \ge \frac{1}{b}\), то \(a \le b\)

Теорема (о 2 городовых)

\(0 \le a_n \le b_n\) - то есть \(0 \le a_1 \le b_1; 0 \le a_2 \le b_2; 0 \le a_3 \le b_3; \cdots\)

А также \(\lim\limits_{n \to \infty} b_n = 0\)

\(\lim\limits_{n \to \infty} a_n = 0\)

Запишем по определению, что \(\lim\limits_{n \to \infty} b_n = 0\). \[\forall \varepsilon \; \exists N \; \forall n > N \; |b_n - 0| < \varepsilon\] Уберем 0 \[\forall \varepsilon \; \exists N \; \forall n > N \; |b_n| < \varepsilon\] Теперь добавим к этому, что \(|a_n| \le |b_n|\)… \[\forall \varepsilon \; \exists N \; \forall n > N \; |a_n| \le |b_n| < \varepsilon\] …и уберем \(|b_n|\) \[\forall \varepsilon \; \exists N \; \forall n > N \; |a_n| < \varepsilon\] Неожиданно мы получили ровно определение, что \(\lim\limits_{n \to \infty} a_n = 0\)

Что и требовалось доказать

Если на нормальном языке, то у нас есть последовательность \(b_n\), которая начиная с некоторого момента будет близко к \(0\). А \(a_n\) меньше чем \(b_n\), то есть еще ближе к \(0\). Значит она тем более стремится к \(0\).

4

Определение, что \(\lim\limits_{n \to \infty} a_n = A\): \[ \forall \varepsilon \; \exists N \; \forall n > N \; |a_n - A| < \varepsilon\]

По определению предела доказать, что \(\lim\limits_{n \to \infty} \frac{1}{n^2} = 0\)

Нам надо доказать, что \(\forall \varepsilon \; \exists N \; \forall n > N \; |\frac{1}{n^2} - 0| < \varepsilon\)

Возьмем и просто найдем это \(N\).

Наша последовательность убывает, так что если мы найдем какой-то \(N\), меньший \(\varepsilon\), все следующие автоматически будут меньше.

Осталось для любого \(\varepsilon\) уметь находить \(N\), что |\(\frac{1}{N^2}| < \frac{1}{n} < \varepsilon\).

Кажется \(N = \sqrt{\frac{1}{\varepsilon}}\) довольно неплохо подходит - если мы его подставим, то мы получим число, меньшее \(\varepsilon\), а все следующие будут еще меньше, потому что последовательность убывает.

По определению предела доказать, что \(\lim\limits_{n \to \infty} \frac{3^{n+1} - 1}{3^n} = 3\)

Нам надо доказать, что \(\forall \varepsilon \; \exists N \; \forall n > N \; |\frac{3^{n+1} - 1}{3^n} - 3| < \varepsilon\)

Также просто найдем этот \(N\).

Преобразуем выражение

\[ |\frac{3^{n+1} - 1}{3^n} - 3| = |3 - \frac{1}{3^n} - 3| = |\frac{1}{3^n}| \]

Осталось найти \(N\), начиная с которого для всех \(n\) будет выполнено \(|\frac{1}{3^n}| < \varepsilon\)

Опять \(\frac{1}{3^n}\) убывает, так что если мы найдем какой-то \(N\), меньший \(\varepsilon\), все следующие автоматически будут меньше.

Осталось для любого \(\varepsilon\) уметь находить \(N\), что |\(\frac{1}{3^N}| < \varepsilon\).

Заметим, что \(3^N \ge N\), значит \(\frac{1}{3^N} \le \frac{1}{N}\) (да, это опять с потолка)

Кажется подходит \(N = \frac{1}{\varepsilon}\). Подставим его, получим \(\frac{1}{3^N} \le \frac{1}{N} = \frac{1}{\frac{1}{\varepsilon}} = \varepsilon\). То есть для \(N = \frac{1}{\varepsilon}\) неравенство \(\frac{1}{3^N} \le \varepsilon\) выполнено. А все следующие \(\frac{1}{3^{N+1}}, \frac{1}{3^{N+2}}, \frac{1}{3^{N+3}}, \cdots\) будут меньше, потому что последовательность убывает.

Найти предел \(\sqrt{n + 1} - \sqrt{n - 1}\). Верь, это \(0\).

Снова преобразуем:

\[\sqrt{n + 1} - \sqrt{n - 1} = \frac{(\sqrt{n + 1} - \sqrt{n - 1}) (\sqrt{n + 1} + \sqrt{n - 1})} {\sqrt{n + 1} + \sqrt{n - 1}} = \] \[ = \frac{(n + 1) - (n - 1)}{\sqrt{n + 1} + \sqrt{n - 1}} = \frac{2}{\sqrt{n + 1} + \sqrt{n - 1}} \]

Заметим, что \(\sqrt{n+1} + \sqrt{n-1}\) это примерно \(2\sqrt{n}\) и оно \(\ge \sqrt{n}\)

Значит \(\frac{1}{\sqrt{n+1} + \sqrt{n-1}} \le \frac{1}{\sqrt{n}}\)

\[ \frac{2}{\sqrt{n + 1} + \sqrt{n - 1}}\le \frac{2}{\sqrt{n}}\]

То есть \(\frac{2}{\sqrt{n + 1} - \sqrt{n - 1}} \le \frac{2}{\sqrt{n}}\). Но \(\frac{2}{\sqrt{n}} \to 0\), значит \(\frac{2}{\sqrt{n + 1} - \sqrt{n - 1}} \to 0\) тоже. (смотри теорему в самом начале)

Найти предел \(\frac{4n + 1}{4n! + (n - 1)!}\)

  • \(4n + 1 \le 8n\)

  • \(4n! + (n-1)! \ge n^2\) - (я верю…), значит \(\frac{1}{4n! + (n-1)!} \le \frac{1}{n^2}\)

Отсюда получаем \(\frac{4n + 1}{4n! + (n - 1)!} \le \frac{8n}{4n! + (n - 1)!} \le \frac{8n}{n^2} = \frac{8}{n} \to 0\), значит \(\frac{4n + 1}{4n! + (n - 1)!} \to 0\)

8 Коши

\(\sum (\frac{2n}{3n + 5})^n\)

\[a_n = (\frac{2n}{3n + 5})^n\] \[\sqrt[n] {a_n} = \frac{2n}{3n + 5}\] Это выражение \(\to \frac{2}{3}\), значит ряд сходится

\(\sum (1 + \frac{1}{n})^{n^2}\)

\[a_n = (1 + \frac{1}{n})^{n^2}\] \[\sqrt[n] {a_n} = (1 + \frac{1}{n})^n\] Этот ряд сходится к числу \(e = 2.71...\), как оказалось, про этот ряд было на лекции, так что ты можешь сказать, что корень сходится к \(e > 1\), значит ряд расходится

9 Даламбер

\(\sum \frac{n + 1}{n!}\)

\[a_n = \frac{n + 1}{n!}\] \[\frac{a_{n+1}}{a_{n}} = \frac{\frac{n + 2}{(n + 1)!} }{\frac{n + 1}{n!} } = \frac{(n + 2)n!}{(n + 1)!(n + 1)} = \frac{n + 2}{(n + 1)^2} \to 0\]

Значит сходится

\(\sum \frac{5^n}{n^5}\)

\[a_n = \frac{5^n}{n^5}\] \[\frac{a_{n+1}}{a_{n}} = \frac{\frac{5^{n+1}}{(n+1)^5} }{\frac{5^n}{n^5} } = \frac{5^{n+1}n^5}{5^n(n+1)^5} = 5 (\frac{n}{n+1})^5\]

Давай еще раз поверим, что \((\frac{n}{n+1})^5\) стремится к 1 и \(5 (\frac{n}{n+1})^5\) стремится к \(5\), значит ряд расходится
