6. Коши

\[\lim_{x \to x_0} f(x) = A\]

Надо для любого \(\varepsilon\) уметь находить область (=окрестность) вокруг \(x_0\), для любого \(x\) из которой \(|f(x) - A| < \varepsilon\).

Надо для любого отрезка с центром \(L\) уметь находить отрезок с центром \(x_0\), что если ты из этого отрезка возьмешь точку \(x\), то \(f(x)\) будет в том отрезке, содержащем \(L\).

Также как с пределами, научимся получать по \(\varepsilon\) эту окрестность.

В данном случае \(x_0 = 0\), \(f(x) = |x|\), \(A = 0\).

Тогда надо по \(\varepsilon\) находить область вокруг \(x_0 = 0\), для любого \(x\) из которой \(||x| - 0| < \varepsilon\).

\(||x| - 0| < \varepsilon\) это \(|x| < \varepsilon\). Если теперь взять отрезок \((-\varepsilon; \varepsilon)\), то это будет окрестность точки \(x_0 = 0\), для каждого \(x\) из которой \(|x| < \varepsilon\)

8. Разложения

Знаем, что \[ e^x = 1 + \frac{x}{1!} + \frac{x^2}{2!} + \dots = \sum_{i = 0}^{\infty} \frac{x^i}{i!} \] Хотим \(\frac{e^x + e^{-x}}{2}\).

Подставив \(-x\) получаем \[ e^{-x} = 1 + \frac{-x}{1!} + \frac{(-x)^2}{2!} + \frac{(-x)^3}{3!} + \dots = 1 - \frac{x}{1!} + \frac{x^2}{2!} - \frac{x^3}{3!} + \dots = \sum_{i = 0}^{\infty} \frac{(-1)^i x^i}{i!} \]

Это то же разложение \(e^x\), только теперь слагаемые на нечетных местах с минусом.

Найдем \(e^x + e^{-x}\). \[e^x + e^{-x} = \sum_{i = 0}^{\infty} (\frac{x^i}{i!} + \frac{(-1)^i x^i}{i!}) \]

На нечетных индексах получим разность одинаковых слагаемых, на четных сумму одинаковых слагаемых. В итоге \[ \sum_{i = 0}^{\infty} (\frac{x^i}{i!} + \frac{(-1)^i x^i}{i!}) = \sum_{i - четный, i \ge 0}^{\infty} 2\frac{x^i}{i!} = \sum_{k = 1}^{\infty} 2\frac{x^{2k}}{(2k)!}\]

После деления на \(2\) получим

\[ \sinh{x} = \frac{e^x + e^{-x}}{2} = \sum_{k = 0}^{\infty} \frac{x^{2k}}{(2k)!} \]

Аналогично \[ \cosh{x} = \frac{e^x - e^{-x}}{2} = \sum_{k = 0}^{\infty} \frac{x^{2k+1}}{(2k+1)!} \]

Разложить \(\frac{1}{x} \sin x^2\)

\[ \sin x = \frac{x}{1!} - \frac{x^3}{3!} + \frac{x^5}{5!} + \dots \]

\[ \sin x^2 = \frac{x^2}{1!} - \frac{x^6}{3!} + \frac{x^{10}}{5!} + \dots \]

\[ \frac{1}{x} \sin x^2 = \frac{x}{1!} - \frac{x^5}{3!} + \frac{x^{9}}{5!} + \dots \]

Нам надо до \(x^5\), тогда \[ \frac{1}{x} \sin x^2 = \frac{x}{1!} - \frac{x^5}{3!} \]
