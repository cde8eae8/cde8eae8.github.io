1. \(f(x) = \sqrt{1 + x^2}\)

  1. \(f(0) = \sqrt{1 + 0^2} = \sqrt{1} = 1\)

  2. \(f(x^2) = \sqrt{1 + (x^2)^2} = \sqrt{1 + x^4}\)

  3. \(f(-x) = \sqrt{1 + (-x)^2} = \sqrt{1 + x^2}\)

  4. \(f(\frac{1}{x}) = \sqrt{1 + (\frac{1}{x})^2}\)

  5. \(\frac{1}{f(x)} = \frac{1}{\sqrt{1 + x^2}}\)

  6. \(f(a) + 1 = \sqrt{1 + a^2} + 1\)

2. Область определения (при каких \(x\) имеет смысл выражение)

\(y = \frac{3x+1}{x^2-1}\)

Единственное условие - знаменатель не равен \(0\). \[x^2 - 1 \neq 0 \Leftrightarrow x \neq \pm 1\] Область определения - любой \(x\) кроме \(\pm 1\)

\(y = \frac{\ln(x)}{\sqrt{|x^2 - 2|}}\)

Ограничения:

  1. Выражение под корнем больше или равно 0

  2. Знаменатель не равен \(0\)

  3. Аргумент логарифма больше \(0\)

Считаем:

  1. \(|x^2 - 2| \ge 0\) выполнено всегда

  2. Знаменатель \[\begin{align} & \sqrt{|x^2 - 2|} \neq 0 \\ & |x^2 - 2| \neq 0 \\ & x^2 - 2 \neq 0 \\ & x \neq \pm \sqrt{2} \end{align}\]

  3. Логарифм: \(x > 0\)

В итоге получаем, \(x > 0\) и \(x \neq \sqrt{2}\)

\(y = \sqrt{-x} + \tan(x)\)

  1. Выражение под корнем должно быть неотрицательным

  2. Аргумент тангенса не должен быть равен \(\frac{\pi}{2} + \pi n, n \in \mathbb{Z}\)

Считаем:

  1. Корень \[\begin{align} & -x \ge 0 \\ & x \le 0 \end{align}\]

  2. Тангенс: \(x \neq \frac{\pi}{2} + \pi n, n \in \mathbb{Z}\)

В итоге \(x \neq \frac{\pi}{2} + \pi n, n = -1, -2, \dots\)

\(y = \arccos \frac{2x}{1 + x}\)

  1. Аргумент арккосинуса должен быть от \(-1\) до \(1\)

  2. Знаменатель не должен быть равен \(0\)

Считаем:

  1. \[\begin{align} & -1 \le \frac{2x}{1 + x} и \frac{2x}{1 + x} \le 1 \\ & 0 \le \frac{3x + 1}{1 + x} и \frac{x - 1}{1 + x} \le 0 \\ \end{align}\] Хотим числитель и знаменатель одного знака в первом случае и разных во втором.

В первом случае это получается при \(x \ge -\frac{1}{3}\) (оба положительны) или \(x \le -1\) (оба отрицательны). Во втором при \(-1 \le x \le 1\)

Оба условия выполняются при \(x = -1, -\frac{1}{3} \le x \le 1\)

  1. Можно было отсечь в прошлом пункте, \(x \neq -1\), так как знаменатель обращается в 0.

В итоге \(-\frac{1}{3} \le x \le 1\)

Область значения (какие y могут быть)

\(y = x^2 - 8x + 20\)

Преобразуем:

\[y = x^2 - 8x + 20 = x^2 - 2 \cdot 4 \cdot x + 4^2 + 4 = (x - 4)^2 + 4\]

Так как \((x - 4)^2\) принимает значение от \(0\) до \(\infty\), \(y\) принимает значения от \(4\) до \(\infty\)

\(y = \frac{1}{x} + 4\)

\(\frac{1}{x}\) принимает все значения кроме \(0\), значит \(y\) принимает все значения кроме \(4\).

\(y = 2 \sin x - 7\)

\[\begin{align} & -1 \le \sin x \le 1 \\ & -2 \le 2 \sin x \le 2 \\ & -9 \le 2 \sin x - 7 \le -5 \end{align}\]

\(y = \arcsin \frac{x^2 + 1}{2x}\)

Давайте посмотрим, при каких \(x\) эта функция существует.

\[\begin{align} & -1 \le \frac{x^2 + 1}{2x} и \frac{x^2 + 1}{2x} \le 1 \\ & 0 \le \frac{x^2 + 2x + 1}{2x} и \frac{x^2 - 2x + 1}{2x} \le 0 \\ & 0 \le \frac{(x + 1)^2}{2x} и \frac{(x - 1)^2}{2x} \le 0 \\ \end{align}\] Первое выполнено при \(x = -1, x \ge 0\), второе при \(x = 1, x \le 0\), а также \(x \neq 0\) из-за знаменателя. Получаем что функция вообще существует только при \(x = \pm 1\), отсюда просто подставив получим чему может быть равен \(y\): \(\arcsin \frac{(1^2) + 1}{2 \cdot 1} = \arcsin 1 = \frac{\pi}{2}\), \(\arcsin \frac{((-1)^2) + 1}{2 \cdot (-1)} = \arcsin -1 = -\frac{\pi}{2}\).

Чтобы их найти достаточно выразить \(x\) из \(y = f(x)\).

Выразим \(x\) из \(y = \sqrt{x}\). \(x = y^2\)

Значит обратная функция \(g(x) = x^2\), проверим: \(g(f(x)) = (\sqrt{x})^2 = x\)

\(f(x) = \frac{2}{x + 3}\)

Выразим \(x\) из \(y = \frac{2}{x + 3}\). \(x = \frac{2}{y} - 3\).

Проверка аналогична

\(f(x) = \arctan(\ln 3x)\)

Аналогично \[\begin{align} & y = \arctan(\ln 3x) \\ & \tan y = \ln 3x \\ & e^{\tan y} = 3x \\ & \frac{e^{\tan y}}{3} = x \end{align}\]

6. Коши

Надо для любого \(\varepsilon\) уметь находить область (=окрестность) вокруг \(x_0\), для любого \(x\) из которой \(|f(x) - A| < \varepsilon\).

Также как с пределами, научимся получать по \(\varepsilon\) эту окрестность.

В данном случае \(x_0 = 0\), \(f(x) = |x|\), \(A = 0\).

Тогда надо по \(\varepsilon\) находить область вокруг \(x_0 = 0\), для любого \(x\) из которой \(||x| - 0| < \varepsilon\).

\(||x| - 0| < \varepsilon\) это \(|x| < \varepsilon\). Если теперь взять отрезок \((-\varepsilon; \varepsilon)\), то это будет окрестность точки \(x_0 = 0\), для каждого \(x\) из которой \(|x| < \varepsilon\)
